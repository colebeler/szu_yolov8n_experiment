# szu-yolov8n-experiment
完善中
## 项目名称
深圳大学电子与信息工程学院《人工智能》课程实验：基于YOLOv8n的道路场景行人与车辆目标检测

---

## 一、项目背景与实验目标
目标检测是计算机视觉领域的核心任务，广泛应用于智能交通、自动驾驶、安防监控等场景。本次实验旨在通过实践掌握深度学习目标检测算法的完整开发流程，具体目标如下：
1.  理解YOLO系列目标检测算法的基本原理，掌握YOLOv8n轻量化模型的结构特点与优势；
2.  熟练使用Anaconda搭建Python开发环境，掌握Ultralytics框架的基础配置与使用；
3.  掌握目标检测数据集的构建流程，学会使用LabelImg工具进行手工标注，理解YOLO格式数据集的规范；
4.  完成模型训练、参数调优与结果评估，掌握mAP、损失函数等关键指标的分析方法；
5.  实现模型推理验证，检验模型在实际场景中的检测效果，复现完整的目标检测流程。

---

## 二、实验环境与准备工作
### 1. 开发与硬件环境
- **操作系统**：Windows 10/11
- **开发语言**：Python 3.8
- **核心框架**：Ultralytics YOLOv8（版本：8.4.21）
- **开发工具**：Anaconda、PyCharm、LabelImg标注工具、LaTeX编辑器
- **硬件支持**：NVIDIA GPU，满足模型训练的算力需求

### 2. 数据集构建
本次实验数据集通过pexels平台收集道路场景图片，筛选包含行人、车辆目标的图像共230张
- **数据标注**：使用LabelImg工具完成手工标注，标注格式为YOLO标准，生成对应的`.txt`标注文件，包含目标类别ID、归一化坐标信息；
- **目标类别**：仅包含两类目标——`person`（行人）与`car`（车辆）；
- **配置文件**：编写`data.yaml`文件，定义训练集/验证集路径、目标类别数量与名称，完成数据集结构规范。

---

## 三、模型训练与配置
### 1. 模型选择
本次实验选用YOLOv8n模型，该模型为YOLOv8系列的轻量化版本，在保持较高检测精度的同时，具备更快的推理速度，适合课程实验场景，兼顾训练效率与检测效果。

### 2. 训练参数配置
- 输入图像尺寸：640×640
- 批处理大小（batch size）：8
- 训练轮次（epochs）：50
- 优化器：Adam优化器
- 早停机制（patience）：50，防止模型过拟合
- 预训练权重：加载YOLOv8n官方预训练权重，基于自定义数据集进行微调训练

### 3. 训练结果与指标分析
训练过程中实时输出box_loss、cls_loss等损失值及mAP指标变化，训练完成后关键指标表现良好：
- **mAP@0.5**（IoU阈值为0.5时的平均精度）：达到0.87795，表明模型对两类目标的检测精度较高；
- **mAP@0.5:0.95**（IoU阈值0.5-0.95的平均精度）：达到0.58119，模型在不同IoU阈值下的鲁棒性良好；
- **损失函数**：box_loss（边界框损失）与cls_loss（分类损失）随训练轮次持续下降并趋于稳定，无明显震荡，模型收敛效果良好，未出现过拟合现象。

---

## 四、模型推理与结果验证
训练完成后，使用模型的最优权重文件（`best.pt`）编写推理脚本，对测试图片进行目标检测验证：
- 脚本支持单张图片推理，可自动加载权重文件，对输入图片进行目标检测；
- 输出内容包含检测结果图像（标注目标类别、置信度与检测框），同时在控制台打印检测结果信息，包括目标类别、置信度、坐标等；
- 推理结果显示，模型能够准确识别场景中的行人与车辆目标，置信度普遍较高，检测框位置贴合目标，无明显漏检、误检情况，验证了模型的实际检测效果。
